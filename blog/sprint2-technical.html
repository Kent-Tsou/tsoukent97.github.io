Margin, Border, and Padding?



Margin


The space between the border and the next element of the design is called margin. E.g., the space outside the border and between it and the other elements. Margin surrounds all four sides of the content which is available for change. When creating the gap, the padding either grows the element’s size or shrinks the content inside. By default, the size of the element increases.


How to create a margin?


            
                p {margin: 25px 50px 75px 100px}
                p {margin-top: 100px;
                   margin-bottom: 100px;
                   margin-right: 150px;
                   margin-left: 80px;}

Above we have two examples on how to adjust margins. The first example is a longer way of sepcifying 4 sides for the margin, by listing them individually. The second example is the shorter version, which have 100, 100, 150 and 80 corresponding to top, bottom, left and right respectively.


Border


CSS border properties allow you to specify the style, width and the color of an element’s border. The “border-style” property includes dotted, dashed, solid, double, groove, ridge, inset, outset, none, hidden and much more. Each of them presenting different types of borders.


How to create a border?

        
            body{border: 3px solid #042331;}

The example above shows how we want a solid border to wrap arond our body with a thickness of 3 screen pixels and with color #042331 applied to it.



Padding


The space that’s inside the element between the element and the border. Same as margin, padding goes around all four sides of the content and you can change each side too. When changing the padding, you want to create space between an element and the edge of the container, or the border. You could also change the size of an element with adjusting the padding.


How to create a padding?

            
                body{padding: 35px 70px 50px 90px;}

This example above shows that we creating padding for the body with 35, 70, 50 and 90 corresponding to top, right, left and bottom respectively.
